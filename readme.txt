This plugin helps you to do the HTTPCS validation process automatically.

== Description ==
This plugin helps you to do the HTTPCS validation process automatically. It will contact the HTTPCS validation service to get a validation file which will help us to validate your website automatically. Also, you will be able to signup at HTTPCS.com if you don't have an account yet or to login if you already own an HTTPCS acccount. [HTTPCS](https://www.httpcs.com/ "European cybersecurity leader")

== Frequently Asked Questions ==
= Why should I use the HTTPCS plugin ? =
The HTTPCS plugin helps you to create an HTTPCS account if you don't own one yet (or to login if you're already registered at HTTPCS.com) and also to validate your website automatically. It will create a validation file on your server which will give us the opportunity to validate your website.

= How does it work ? =
The plugin will get a file which is related to your website and your account to validate your website. If a problem occurs, please check the rights of your server. You will maybe need to make file manually.
